•  What is a gitignore file and what should it include for a Java project? 
	In a Git repository, a.gitignore file is a configuration file that lists the files or folders that Git should not track.
	For a Java project, typically following files and directories included in a .gitignore file:
	Binaries generated by the build process
	IDEs and build tools metadata
	Log files
	Operating system generated files

•  What does fast-forward in git mean? 
	A "fast-forward" in Git is a straightforward branch pointer shift to a fresh commit without the need for a merging commit.

•  Why should you have different branches? 
	You can work on numerous distinct versions of a codebase simultaneously by using different branches in Git.

•  In a project how many remote Git repositories do you usually have? 
	It is dependent upon the project's size and structure, as well as its collaboration and deployment procedures. 
	To effectively control the flow of changes and enforce security regulations, larger projects or companies might employ many remote repositories.
	The usage of numerous remote repositories in some projects is also possible to support various deployment environments or stages of the software development life cycle.
